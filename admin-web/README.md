# Novel Admin Web

Background management system for NovelVoiceApp.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start Development Server:
   ```bash
   npm run dev
   ```

## Features
- Novel Management
- System Management (Roles, Menus)

## Backend Requirements
- Ensure backend is running on port 8080.
- Execute the updated `schema.sql` to create `sys_role`, `sys_menu` tables.
